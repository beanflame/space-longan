# XCache

XCache 是2021年12月24日开始，一门免费开源的轻量级的 C/C++ 跨平台集成开发环境，由 SpaceLongan 主导开发。

打开文件速度快，并且稳定性良好，快速编写代码，支持多人在线编辑，插件，Git仓库。

在所有主流电脑平台（OSX、Windows 和 Linux）都能支持运行。
